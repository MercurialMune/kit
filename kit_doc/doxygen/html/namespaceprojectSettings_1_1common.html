bool projectSettings.common.CHECK_IF_WEB_SITE_DISABLED = True

[ bool ] - Whether to check web site is disabled.

If this variable is True, projectSettings.models.ProjectSettings.getIsWebSiteEnabled method will be invoked in core.middlewares.IsWebSiteEnabledMiddleware middleware. If web site is disabled request will be redirected to projectSettings.common.WEB_SITE_DISABLED_REDIRECT_URL
